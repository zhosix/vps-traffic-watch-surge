# VPS Traffic Watch Surge

Remote module:

```text
https://raw.githubusercontent.com/zhosix/vps-traffic-watch-surge/main/traffic-watch.sgmodule
```

Arguments:

- `BASEURL`: your VPS Traffic Watch backend, for example `https://watch.example.com`
- `TOKEN`: `apiToken` or `surge.sharedSecret`
- `DEVICE`: device name shown in the dashboard
- `PANELINTERVAL`: Surge panel refresh interval in seconds
