# VPS Traffic Watch Surge

Remote module:

```text
https://raw.githubusercontent.com/zhosix/vps-traffic-watch-surge/main/traffic-watch.sgmodule
```

Arguments:

- `BASE_URL`: your VPS Traffic Watch backend, for example `https://watch.example.com`
- `API_TOKEN`: `apiToken` or `surge.sharedSecret`
- `SURGE_DEVICE`: device name shown in the dashboard
- `PANEL_INTERVAL`: Surge panel refresh interval in seconds
- `HEARTBEAT_CRON`: cron expression for heartbeat reports
